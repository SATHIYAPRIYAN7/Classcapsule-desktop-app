For Windows Users:

  • Ensure your audio drivers are properly installed and working
  • Check that your microphone is not muted in Windows settings
  • Allow screen sharing when prompted by the browser
  • If audio fails, try using only screen audio (disable microphone)

For Mac Users:

  • Go to System Preferences > Security & Privacy > Privacy > Screen Recording
  • Add Chrome/Edge to the list of allowed applications
  • Go to System Preferences > Security & Privacy > Privacy > Microphone
  • Add Chrome/Edge to the microphone permissions list
  • Restart your browser after granting permissions

General Tips:

  • Use Chrome or Edge browser for best compatibility
  • Ensure you have a stable internet connection
  • Close other applications that might be using audio/video
  • Try refreshing the page if permissions are denied
